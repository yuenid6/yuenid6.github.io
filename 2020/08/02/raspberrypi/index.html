树莓派安装系统与无屏幕下安装屏幕驱动

八月 02, 2020
  • 树莓派

所需材料我就不说了
大致流程

  1. 先去官网下载树莓派官方系统，并将系统烧录到Micro SD卡中：
  2. 将SD卡插入树莓派接入电源、鼠标、键盘……….
  3. 进行相关配置
    一 烧录
    先去官网下载官方系统，有下面三个版本

    Lite版本是最小化安装，没有桌面环境；Desktop版本则带有桌面； Desktop and recommended software版本还带有推荐软件，但比较大。我选择了最完整的版本，点Download ZIP按钮下载压缩包并解压，得到扩展名为.img的系统镜像文件。不放心的话可以自己SHA256一下。
    准备好Micro SD卡和读卡器，先用SDFormatter格式化Micro SD卡

    然后用Win32 Disk Imager将解压出的Raspbian系统镜像文件写入Micro SD卡，树莓派的系统盘就做好了。

    注意：系统写完后，如果弹出来要格式化剩余空间的盘，一定不要格式化！！！不然可能会遇到树莓派开机时，系统无法启动的问题！这是因为树莓派系统启动时默认只使用操作系统需要的SD卡空间。这意味着即使你有一个很大容量的SD卡，操作系统也不会使用那么多容量。如果系统启动报错： end Kernel panic - not syncing:vfs unable to mount root fs on unknown -block(179,2)，则可能是你在SD卡中做完系统格式化剩余盘导致的，不要选择格式化，再插入树莓派中重新启动即可。

    现在我们安装好了系统

    安装屏幕

    我们需要一根网线和工具Putty
    将网线插入树莓派在路由器管理界面(例如192.1681.1)查看树莓派的IP地址

    填入树莓派的IP地址点击Open就可以连接到树莓派了不过只能控制终端并不是远程桌面
    在终端输入 用来下载驱动，输入ls来查看在目录中是否存在LCD-show这个文件
    输入chmod -R 755 LCD-show给该文件夹可以读取修改的权限，再输入cd LCD-show/进入到LCD-show这个目录。

    再输入下面面安装驱动的语句进行安装，安装完后会自动重启。
    #HDMI显示屏驱动（系统的默认驱动是这个）:
    sudo ./LCD-hdmi
    #3.5寸树莓派 LCD显示屏驱动:
    sudo ./LCD35-show
    #5寸树莓派LCD显示屏驱动:
    sudo ./LCD5-show
    #7寸800X480显示屏驱动:
    sudo ./LCD7B-show
    #7寸1024X600显示屏驱动:
    sudo ./LCD7C-show
    #2.4寸树莓派 LCD显示屏驱动:
    sudo ./LCD24-show
    #2.8寸树莓派 LCD显示屏驱动:
    sudo ./LCD28-show
    #3.2寸树莓派 LCD显示屏驱动:
    sudo ./LCD32-show
    #3.97寸树莓派 LCD显示屏驱动:
    sudo ./LCD397-show
    #4.3寸树莓派 LCD显示屏驱动:
    sudo ./LCD43-show
